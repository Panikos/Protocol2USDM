# Utilities

Setup and utility scripts for Protocol2USDM.

## Files

| File | Purpose |
|------|---------|
| `setup_google_cloud.ps1` | PowerShell script to configure Google Cloud credentials |

## Usage

### Google Cloud Setup (Windows)

```powershell
# Run setup script
.\utilities\setup_google_cloud.ps1
```

This configures:
- Google Cloud project
- Vertex AI credentials
- Application default credentials
